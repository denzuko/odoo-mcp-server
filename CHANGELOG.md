## [1.0.0] ‚Äî 2026-04-29

### Added

- `main.c` ‚Äî kcgi HTTP server, `/mcp` (MCP JSON-RPC 2.0) +
  `/healthz`. FastCGI + plain CGI modes. Native target only.
- `mcp.c` / `mcp.h` ‚Äî MCP protocol: `initialize`, `tools/list`,
  `tools/call` dispatch for all four Odoo tools.
- `odoo.c` / `odoo.h` ‚Äî Odoo XML-RPC client: `authenticate` +
  `execute_kw`. Hand-rolled XML-RPC envelope builder, response
  parser, and JSON‚ÜíXML-RPC converter.
- `net.c` / `net.h` ‚Äî Transport abstraction. The single `#ifdef`
  boundary: BSD sockets + libtls (native) vs. `http_fetch` import
  from CF Worker JS shim (`__wasm__`).
- `json.h` ‚Äî Arena-backed JSON builder + recursive-descent parser.
- `arena.h` ‚Äî Bump-pointer arena allocator (tsoding pattern).
- `sv.h` ‚Äî String view, zero-copy (tsoding pattern).
- `config.h` ‚Äî `ODOO_*` env var binding, fail-fast on missing vars.
- `nob.c` ‚Äî Build driver. `./nob` ‚Üí native ELF, `./nob wasm` ‚Üí
  `wasm32-wasi` module, `./nob clean`.
- `nob.h` ‚Äî Vendored tsoding/nob.h v3.8.2.
- `workers/worker.js` ‚Äî 15-line CF Workers JS shim. Loads
  `odoo-mcp.wasm`, bridges `http_fetch` import to Workers `fetch()`.
- `policy/c_quality.rego` ‚Äî OPA gate over cppcheck + clang-tidy
  SARIF. Blocks: null-deref, buffer OOB, UAF, uninit reads,
  format-string injection, banned APIs, style violations.
- `policy/sarif.rego` ‚Äî OPA CVE gate over osv-scanner SARIF.
  Blocks CVSS ‚â• 7.0.
- `docs/index.html` ‚Äî GitHub Pages site. Da Planet design system
  (Share Tech Mono / Rajdhani, canonical tokens from
  thelounge-theme-daplanet v1.0.8).

## [1.4.0] ‚Äî 2026-04-29

### Added

- `containers/odoo-mcp.container` ‚Äî rootless Podman Quadlet unit.
  Mirrors fts-quadlet-setup conventions. Binds to 127.0.0.1:8000,
  HAProxy routes mcp.dapla.net/* to it. AutoUpdate=registry.
  DropCapability=ALL + NoNewPrivileges=true security posture.
- `networks/odoo-mcp.network` ‚Äî isolated bridge (10.89.3.0/24).
- `volumes/odoo-mcp-data.volume` ‚Äî named volume for runtime state.
- `env/odoo-mcp.env` ‚Äî template only; live copy generated by setup script.
- `odoo_mcp_setup.sh` ‚Äî POSIX sh installer (mirrors fts_setup.sh).
  Creates odoo-mcp service account (uid 2002), enables linger,
  installs quadlet units, generates env file with real credentials
  (idempotent ‚Äî preserves operator edits on re-run), smoke-tests /healthz.
- `examples/haproxy-odoo-mcp.cfg` ‚Äî HAProxy backend stanza for native deploy.
- `docs/secrets.md` ‚Äî complete secrets reference: GitHub Actions secrets,
  Cloudflare API token permissions, server-side env file, Terraform Worker
  secrets, what is never committed.

## [1.5.0] ‚Äî 2026-04-29

### Fixed

- `nob.c` ‚Äî complete rewrite against real nob.h v3.8.2 API:
  - `nob_cmd_run(&cmd)` replaces deprecated `nob_cmd_run_sync(cmd)`
  - `nob_cmd_append(cmd, ...)` variadic ‚Äî no trailing NULL
  - `nob_temp_sprintf()` for path string construction (replaces
    hallucinated `NOB_CONCAT` macro which does not exist)
  - `#define _POSIX_C_SOURCE 200809L` required by nob.h timespec usage
  - Target binary renamed `odoo-mcp-server` (was `odoo-mcp`)
  - WASM target renamed `odoo-mcp-server.wasm`

### Added

- `Dockerfile` ‚Äî multi-stage build. Stage 1: gcc + kcgi from source +
  nob build driver compiles the C binary. Stage 2: distroless/cc
  final image with only the binary and required shared libs.
  Tagged and pushed to `ghcr.io/denzuko/odoo-mcp-server` by CI.
  This is what the quadlet container unit pulls at runtime.
- CI `docker` job ‚Äî builds and pushes to GHCR on every push to main
  and on semver tags. `GITHUB_TOKEN` permissions: packages:write.
  Tags: `latest` (main only), `v{version}`, `sha-{short}`.
  Docker layer cache via `cache-from/to: type=gha`.

## [1.7.0] ‚Äî 2026-04-29

### Changed

- JSON parsing replaced with rxi/sj.h (public domain, v0.4, 155 lines).
  sheredom/json.h was rejected (malloc-based, breaks arena security model).
  Our own recursive-descent parser in json.h also removed.
  sj.h is a zero-allocation cursor: sj_Reader/sj_Value hold pointers
  into the original buffer ‚Äî no heap, no arena nodes, no copies.

- json.h now contains only the JsonBuf builder (arena-backed, unchanged).
  Parser types (JsonVal, JsonPair, JsonParser, json_parse, json_get) removed.

- mcp.c rewritten to use sj_reader/sj_read/sj_iter_object/sj_iter_array.
  SJ_IMPL defined here (one translation unit owns the implementation).

- odoo.c JSON‚ÜíXML-RPC converter rewritten to use sj.h cursor traversal.
  json_to_xmlrpc() now walks the JSON with sj_iter_object/sj_iter_array
  and emits XML-RPC incrementally into arena-allocated strings.

### Added

- sj.h ‚Äî vendored rxi/sj.h v0.4 (public domain).
  Added to LICENSE.txt vendored section.
